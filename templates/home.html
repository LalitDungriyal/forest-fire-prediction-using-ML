Introduction

Forest-fire-prediction is a web based application developed using python.It allows the user the predict the occurance of the fire in a forest on the basis of various vegetation indices like NDVI, LST, Area. The user needs to be registered in order to predict the occurance of the fire i.e. only authorized users can predict the occurance of fire.

How it works..?

Machine learning is learning from data to be able to predict the future.Hence, we are going to model some parameters crucial for any forest fire to take place and predict the possibility of a forest fire taking place based on that. This project uses the vegetation indices like NDVI(Normalized Difference Vegetation Index), LST(Land Surface Temperature) and Area as the features and fourth column as label which contains the value 0 or 1(not_fire or fire). This model is trained using the Logistic regression technique in machine learning in order the predict the occurance of fire.

Learning Algorithm

This particular problem comes under the category of Supervised Learning. We train our machine learning mode using the following 3 learning models and compare the accuracies:

  1. Linear Regression
  2. Logistic Regression
  3. Support Vector Machine
